## Python vs JavaScript 对比

<PythonEditor compare title="Python vs JavaScript 对比">
```python !! py
# Python 字典操作
person = {"name": "李四", "age": 30, "city": "北京"}
print(f"姓名: {person['name']}")
print(f"年龄: {person['age']}")

# 列表推导式
numbers = [1, 2, 3, 4, 5]
evens = [x for x in numbers if x % 2 == 0]
print(f"偶数: {evens}")
```

```javascript !! js
// JavaScript 对象操作
const person = {name: "李四", age: 30, city: "北京"};
console.log(`姓名: ${person.name}`);
console.log(`年龄: ${person.age}`);

// 数组过滤
const numbers = [1, 2, 3, 4, 5];
const evens = numbers.filter(x => x % 2 === 0);
console.log(`偶数: ${evens}`);
```
</PythonEditor>
